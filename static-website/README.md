# Boosty - Static Website Files

موقع Boosty الثابت جاهز للرفع على GitHub Pages أو أي خدمة استضافة أخرى.

## ملفات الموقع

### الملفات الأساسية
- `index.html` - الصفحة الرئيسية
- `pricing.html` - صفحة الأسعار الكاملة  
- `style.css` - ملف التنسيقات
- `script.js` - ملف الجافاسكريبت والوظائف

## المميزات المتوفرة

### ✅ المميزات الأساسية
- تصميم متجاوب (Mobile Responsive)
- دعم اللغة العربية والإنجليزية مع RTL
- تبديل العملات (جنيه مصري / ريال سعودي / دولار أمريكي)
- وضع مظلم/فاتح (Dark/Light Mode)
- أقسام الخدمات بالتفصيل
- معلومات الاتصال الكاملة

### ✅ المساعد الذكي (Boosty Bot)
- يجيب على أي سؤال (طبخ، برمجة، ثقافة، إلخ)
- يتكلم باللهجة المصرية الودودة
- يقدم معلومات الأسعار والخدمات
- واجهة محادثة تفاعلية

### ✅ خدمات WhatsApp Sender
- 3 أقسام في كارت واحد:
  - باقات الرسائل (8K, 25K, 60K)
  - أكواد التفعيل (شهري، سنوي، مدى الحياة)
  - الجلسة الخاصة (60 دقيقة عبر AnyDesk)

### ✅ الخدمات الأخرى
- التصميم الجرافيكي (يبدأ من 50 جنيه)
- خطط Canva Pro (شهري، سنوي، مدى الحياة، لوحة موزع)
- صفحة أسعار منفصلة مع جداول تفصيلية

## طريقة الرفع على GitHub Pages

### الخطوة 1: إنشاء Repository
1. اذهب إلى GitHub.com
2. اضغط على "New Repository"
3. اسم الـ Repository: `boosty-website`
4. اختر "Public"
5. اضغط "Create Repository"

### الخطوة 2: رفع الملفات
1. اضغط "uploading an existing file"
2. اسحب جميع الملفات الأربعة:
   - `index.html`
   - `pricing.html`
   - `style.css`
   - `script.js`
3. اكتب رسالة Commit: "Add Boosty website files"
4. اضغط "Commit changes"

### الخطوة 3: تفعيل GitHub Pages
1. اذهب إلى Settings في الـ Repository
2. اسكرول لأسفل إلى "Pages"
3. في Source اختر "Deploy from a branch"
4. في Branch اختر "main"
5. اضغط "Save"

### الخطوة 4: الوصول للموقع
بعد دقائق قليلة، الموقع سيكون متاح على:
`https://[اسم-المستخدم].github.io/boosty-website`
